- `mcl_image_cache.json`: Cache file for image processing.
- `README.md`: Project documentation.
- `requirements.txt`: Python dependencies.
- `runtime.txt`: Python runtime version.

## app/
- `__init__.py`: Package marker.

- `context_manager.py`: Context management logic.
- `database.py`: Database connection and models.
- `main.py`: Application entry point.
- `mcl_image_validator.py`: Image validation logic.
- `models.py`: Pydantic data models.
- `services.py`: Business logic services.
- `vision_assistant.py`: Vision AI integration.
- `documents/`: PDF documents for RAG.

  - `Mcl Mobile App Faq Revised.pdf`

- `data-models-backend.md`: Data schema documentation.
