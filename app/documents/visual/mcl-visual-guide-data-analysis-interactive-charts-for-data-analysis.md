# Interactive Charts for Data Analysis

The data analysis charts provide an interactive means to visualize performance data across different departments and markets. Users can select filters such as Company, Report type, Period, and specific time frames like Year or Quarter. This enables detailed analysis of routine check performances, comparisons between departments, and overall market insights.
