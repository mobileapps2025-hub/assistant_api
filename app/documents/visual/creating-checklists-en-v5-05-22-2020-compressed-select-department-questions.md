# Selecting Department Questions

To create a checklist, first choose a department from the right-hand menu. Once a department is selected, you can pick the questions to include by checking the boxes next to each question. You can organize the order of the questions by using the provided sequence icons. Additionally, the order of departments can be adjusted within the MCL Mobile App.
