# Understanding the Task Table

The Task Table provides a detailed overview of current tasks, displaying information across several columns. Each task has an ID and name, but clicking on it will not open any details. If a photo or PDF is attached, corresponding icons appear, allowing you to open these attachments. A pencil icon enables task editing, while the checklist column shows the originating checklist. Dates such as the start date, due date, and creation date are visible, along with the task's current status and assignment details. Notes can be managed by clicking the eye or plus icons, and tasks can be deleted using the trash icon or selected for bulk actions via checkboxes.
