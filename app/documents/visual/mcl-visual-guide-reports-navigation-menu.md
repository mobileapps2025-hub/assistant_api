# Navigation Menu

The left panel menu allows users to access various sections such as Dashboard, Company, Markets, and Reports. Icons next to each option help identify the sections more easily.

The MCL Web interface features a navigation menu that acts as the primary access point for various functions. Each menu item opens a specific configuration area, such as Markets, Departments, and Security, allowing management of respective tasks like store and department management or user permissions. The top right includes icons like a help button, linking to further user documentation.
