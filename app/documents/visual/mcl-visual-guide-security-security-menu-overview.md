# Security Menu Overview

The Security menu in MCL is an essential tool for managing user access within the system. It provides an overview, allowing users to navigate to either the 'Users' or 'Roles' sub-pages. Selecting 'Users' directs you to the user management page where you can create, edit, or delete user accounts and assign access. Choosing 'Roles' allows you to configure permissions and access rules for different roles.
