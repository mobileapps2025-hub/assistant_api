# Store List Configuration

For users assigned the Store Manager role, there is an additional configuration known as the 'Store list.' This feature allows tasks with no specific assignee or those directed to 'All' to be automatically assigned to the Store Manager, ensuring no task remains unassigned and responsibilities are clear at the store level.
