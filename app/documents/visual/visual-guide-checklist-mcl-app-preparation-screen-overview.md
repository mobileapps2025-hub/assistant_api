# Preparation Screen Overview

The preparation screen in the MCL app is designed to allow users to set up a checklist before execution. Users can select a reason for the checklist, which may be 'Routine' for regular tasks or 'Special' for event-based activities. Additionally, users can choose the departments involved in the checklist and then initiate the process by pressing the 'Start Checklist' button.
