# Managing Checklists in the Interface

The Checklist Management section allows users to view and manage all online and offline checklists. You can add new checklists, view existing ones, or export them for further review. The interface includes columns for the checklist name, availability dates ('From' and 'To'), and recurrence. Additionally, it shows which departments have been assigned using checkmarks. To edit a checklist, click the pencil icon next to it.
