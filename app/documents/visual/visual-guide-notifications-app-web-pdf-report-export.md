# Exporting Reports as PDF

The MCL app allows users to export checklist reports as PDF documents. This can be done by selecting the printer or PDF icon, which generates a PDF version of the current report. The exported document includes detailed information such as the checklist name, format, report ID, start and completion times, the evaluator's name, and the overall score.
