# Task Management Screen

The task management screen allows users to view and filter tasks associated with a selected market. At the top, users can filter tasks using the 'Market Selector' dropdown to limit visibility to tasks from a specific market, or by selecting status buttons ('Open', 'Completed', 'Missed'). There are separate blocks for tasks system-wide, tasks created by the user, and tasks assigned to them. The 'Delete tasks' button enables bulk deletion of tasks that have been selected using checkboxes in the task table below.
