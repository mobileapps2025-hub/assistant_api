# Checklist Overview Navigation

To navigate to the 'Checklists Overview' in MCL Web, select 'Checklists' from the menu. This will open the Checklist Overview where you can create or manage checklists through the Checklist Wizard. This level of navigation shows a hierarchical structure with the dashboard as the parent screen.
