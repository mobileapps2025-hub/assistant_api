# Tasks Tab Dashboard View

Switching to the Tasks tab, highlighted in red, changes the dashboard from the Checklists view to a detailed Tasks graph view. This section displays important task-related charts and statistics for the chosen time frame. Users can analyze data through two graphs, offering insights into assigned tasks and their status based on due dates.
