# Recurring Checklists by Status

This graph displays the status of recurring checklists for each store. Each bar uses colors to represent different statuses: green for completed, red for missed, and yellow for open. Users can click on these bars to access a detailed list of the checklists corresponding to each status. By hovering, you can see the ratio of completed, missed, and open checklists.
