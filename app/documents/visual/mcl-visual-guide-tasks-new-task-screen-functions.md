# New Task Screen Functions

The New Task screen allows you to create tasks tailored to specific stores and assign corrective actions after audit findings. You can set deadlines, provide additional documentation such as photos or PDFs, and distribute tasks to multiple users simultaneously. Additionally, you can create newsletters for all stores.
