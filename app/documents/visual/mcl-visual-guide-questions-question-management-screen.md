# Managing Questions in Checklists

The 'Questions Overview' screen allows you to create, edit, or delete questions used in checklists. To create a new question, click the '+' icon. This opens the Question Editor where you can input the question text, additional information, attach files, and set the question type. Ensure each question is assigned to the correct department to appear in the Checklist Wizard. Use the pencil and trash icons to edit or delete existing questions.
