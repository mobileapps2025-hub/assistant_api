# Checklist Management Icons

Icons provide various management options for checklists. The '+' icon opens the Checklist Wizard for creating a new checklist. The pencil icon allows editing of an existing checklist. To delete, use the trash icon. Checklists can be exported to Excel using the export button for offline storage or review.
