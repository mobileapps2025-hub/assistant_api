# Using Tabs for Comparative Analysis

The Comparative Analysis feature includes tabs like Chart (Markets) and Chart (Departments) to visually compare checklist ratings. The Chart (Markets) tab allows you to evaluate checklist ratings by percentage within a single market. The Chart (Departments) tab lets you examine department ratings either for one department within a store or across all stores.
