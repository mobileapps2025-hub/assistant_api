# Task Assignment Rules

In the MCL app, tasks can be assigned to a specific user or to a market if no individual is selected. If a task is assigned to 'All' or lacks a responsible individual, it will follow store-level assignment rules, provided these are configured. Tasks will be visible in task lists and will sync with the web dashboard.
