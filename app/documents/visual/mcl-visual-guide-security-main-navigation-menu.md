# Main Navigation Menu

Upon logging into the MCL Web UI, users are directed to the Dashboard, which displays key information like completed checklists and task assignments. The main navigation menu, located on the left side and highlighted in red, gives access to all primary management modules such as Company, Security, and Reports. For users on smaller screens, this menu can be accessed by clicking the hamburger icon in the top-left corner.
