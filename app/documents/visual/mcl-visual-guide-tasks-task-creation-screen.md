# Task Creation Screen

The task creation screen lets users manually create new tasks. You can assign tasks to specific markets and users, and add necessary details. Required fields include 'Market' and 'Description'. Optional fields like 'Start/Due Date' and a checkbox for photo/comment requirement also exist.

In the 'Task Information' section, use the buttons labeled 'Markets' and 'Users' to view lists of available markets and users, respectively, for assignment.
