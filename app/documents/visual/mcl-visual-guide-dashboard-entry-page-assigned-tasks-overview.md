# Understanding the Assigned Tasks Overview

The Assigned Tasks section displays a bar chart where each bar represents newly created tasks for a specific market. Different colors within the bars indicate various task origins, such as ad hoc or independent tasks. You can hover over a bar to view the ratio of tasks from each origin. Note that the bars are not interactive and cannot be clicked. Tasks may also be listed according to their due status.
