# Creating a New Tag Category

To create a new tag category in the MCL app, navigate to the Tags section. You can select a company and a tag category from the dropdown menus. To add a new tag category, click on the plus icon on the right side of the interface. This allows for the organization and filtering of data by grouping results beyond standard categories like markets and departments.
