# Edit Role Configuration

To edit a role's configuration, you can toggle its permissions and access rules through the checkboxes. Each feature is controlled by a checkbox that you can check or uncheck, depending on whether you want the role to have that feature. The roles are listed on the left, and permissions such as 'Creation of tasks' and 'Monitoring data' can be adjusted.
