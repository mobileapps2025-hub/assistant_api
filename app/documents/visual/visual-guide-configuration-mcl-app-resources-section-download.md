# Downloading Resources in the MCL App

The Resources section in the MCL app allows you to download PDF files linked to checklist questions. This function ensures that you have access to necessary documents even if your device is offline. To download these resources, simply tap the 'Download' button. Be aware that if the file size is large or the internet connection is slow, the download might take a few minutes.
