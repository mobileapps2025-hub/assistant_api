# Main Navigation Menu

When users log into MCL, they arrive at the Dashboard. The main navigation menu is accessible from the panel on the left side of the screen. It provides links to various management and analysis modules. On smaller screens, the menu is hidden and can be opened by clicking the hamburger icon at the top-left corner.

The main navigation menu in MCL Web serves as the root access point for configuring various system features. Options include managing stores, departments, security roles, checklists, and reports. Users can also manage tasks and photos, and analyze data. The top-right menu provides quick access to help and lock screen features, although the notification bell is currently non-functional.
