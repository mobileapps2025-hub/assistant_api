# Main Navigation Menu

Upon logging into the MCL system, users are directed to the MCL Dashboard. The main navigation menu, highlighted on the left side, provides access to various management and analysis modules. If the navigation panel is not visible, users can reveal it by clicking the hamburger icon in the top-left corner.

The main navigation menu in MCL Web is the central hub for accessing various configuration areas. Key features include management options for Markets, Departments, Users, and Checklists, among others. Icons at the top right provide quick access to user documentation via the Help icon and allow lockscreen mode activation through the Lock icon.
