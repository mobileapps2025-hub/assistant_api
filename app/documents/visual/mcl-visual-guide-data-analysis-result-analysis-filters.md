# Result Analysis Filters

The Result Analysis screen in the Data Analysis tab allows users to filter and compare checklist answer data across different stores. You can adjust filters such as Year, Period, Day, Type of question, Market, and specific Questions to customize the data you review. This helps identify trends, benchmark performance, and spot improvements over time. Use the 'Filter' button to apply these settings.
