# Main Navigation Menu

Upon logging into the MCL system, users are greeted with the MCL Dashboard. The main navigation menu is accessible from the panel on the left, highlighted in a red box. This panel allows users to access various management and analysis modules such as 'Markets' and 'Dashboard.' For users on smaller screens, the navigation menu can be revealed by clicking the hamburger icon in the upper left corner.
