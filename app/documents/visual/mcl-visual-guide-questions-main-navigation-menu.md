# Main Navigation Menu

Upon logging in to the MCL Dashboard, users are presented with a navigation panel on the right side of the screen. This panel provides access to various management and analysis modules available in the MCL system. On smaller screens, the navigation menu is hidden but can be accessed by clicking the hamburger icon at the top-left corner.
