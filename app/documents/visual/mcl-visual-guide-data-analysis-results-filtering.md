# Filter and Analyze Results

To filter your checklist data effectively, use the **Filter** button to apply desired filters, including Year, Period, Day, Options, Type of Questions, Market, and Question. The results table will display data across several columns, such as **Question**, which shows the checklist question text, and **Positive Answers**, **Negative Answers**, and **N.Z. Answers**, which detail response counts for each category within the selected timeframe.
