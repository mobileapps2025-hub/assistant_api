# Data Analysis Sub-Menu

The "Data Analysis Sub-Menu" in the MCL Web allows you to access tools for analyzing checklist results. You can compare ratings across different markets, departments, and users. To access this, select "Data Analysis" from the main navigation menu. This will open the Data Analysis Overview, presenting options for Result Analysis and Comparative Analysis.
