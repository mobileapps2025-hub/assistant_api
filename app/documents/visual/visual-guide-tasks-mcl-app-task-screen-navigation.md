# Navigating the Task Screen

The task screen is the main hub for managing tasks in the app. To search tasks, tap the **Filter** icon located at the top left. This opens the search and filtering options. To create a new task, tap the **Plus** icon at the top right, which opens the Create Task screen. The task list displays all stored tasks, and a bottom navigation bar provides access to other app features.
