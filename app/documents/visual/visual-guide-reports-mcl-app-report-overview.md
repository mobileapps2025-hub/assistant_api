# Understanding the Report Overview

The report screen in the MCL App provides a comprehensive summary of the checklist performed at a store. Key details include the checklist name, format, report ID, start and completion dates, tester's name, and the final rating. This overview helps users quickly access essential performance metrics of the evaluation conducted at a particular store.
