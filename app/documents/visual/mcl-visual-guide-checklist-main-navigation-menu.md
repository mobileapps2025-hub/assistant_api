# Main Navigation Menu

Upon logging into the MCL Dashboard, the main navigation panel is accessible on the left side of the screen. This panel, outlined in a red box, allows users to navigate various modules such as Dashboard, Checklists, Reports, and more. On smaller screens, the navigation menu is accessible via the hamburger icon in the top-left corner.

The main navigation menu in MCL Web acts as the root access point for different configuration areas such as 'Markets', 'Departments', 'Security', and more. Each menu item opens a dedicated configuration area where specific tasks can be managed. The top-right icons provide quick access to user documentation and lockscreen mode.
