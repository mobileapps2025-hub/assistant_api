# Results Comparison Overview

The Results Comparison feature allows users to view a side-by-side performance comparison of all completed checklists over a selected time frame. This is presented as a percentage on a user basis. Filters above the charts enable you to refine the data by Markets, Report type (e.g., Routine Check, Special Inspection), and Period (Day, Week, Month, Quarter, Year). Additionally, if 'Day' is selected as the period, you can specify the particular day.
