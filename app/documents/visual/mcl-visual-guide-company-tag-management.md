# Managing Tags and Categories

The Tags section in the app allows you to manage tags and their categories for markets and checklists. Use the dropdown menu to switch between Market Tags and Checklist Tags. You can view, rename, or delete tags using the icons provided. Additionally, tags can be assigned to existing categories or new tag categories can be created using the appropriate options.
