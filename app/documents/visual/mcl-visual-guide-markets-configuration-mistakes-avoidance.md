# Avoiding Common Configuration Mistakes

To ensure seamless operation, avoid common configuration pitfalls. Assign departments to ensure checklists appear for stores. Properly assign tags to enable market analysis. Ensure department combinations are correct to avoid missing questions or sections, and configure batch-uploaded markets so they function properly in the system.
