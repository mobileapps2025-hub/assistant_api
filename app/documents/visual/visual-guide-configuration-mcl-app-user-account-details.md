# User Account Details and Settings

The interface allows users to view essential account details, including the full name of the logged-in user, their username, and the company they are associated with. Users can also see the version of the installed app. To enhance accessibility, the Language Selection feature provides options to switch between German, English, and Spanish. For security, a Log Out Button is available for users to exit their accounts safely.
