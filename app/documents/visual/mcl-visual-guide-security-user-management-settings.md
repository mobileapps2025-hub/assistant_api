# User Management Settings

The User Management section in MCL allows you to specify essential details needed for user creation, such as username and role, which are mandatory fields. Passwords are required conditionally, especially when adding new users. You can also add secondary and tertiary email addresses for notifications. Assigning a role like Admin, Store Manager, or Team Member determines the access and permissions within the MCL system.
