# New and Due Tasks Notification Dialog

When opening the MCL app, a dialog appears to inform users about tasks that are new or due today. The dialog has a title labeled 'Tasks' (AUFGABEN) and messages indicating the number of new tasks and tasks due. Users can choose to 'Cancel' the notification, which dismisses it, or select 'View' to open the task list. This notification appears during task creation, due dates, or at specified system times.
