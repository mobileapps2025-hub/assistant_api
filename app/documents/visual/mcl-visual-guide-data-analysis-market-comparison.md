# Market Comparison

The Market Comparison feature in the Data Analysis section allows users to view a side-by-side comparison of market performances in percentage. Users can select specific markets and other parameters such as company, report type, and period to generate an interactive chart for comparative analysis. This feature aids in evaluating how different markets perform relative to each other over selected timeframes.
