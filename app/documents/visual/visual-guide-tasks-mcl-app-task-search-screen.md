# Task Search Screen

The Task Search screen enables users to locate tasks by applying specific filters. You can refine your search by adjusting the date range with an option to select which task date type to consider: Due, Created, or Started. Additional filters include Market, Assigned user, and Status. Once you've set your parameters, press the 'SEARCH' button to find the desired tasks.
