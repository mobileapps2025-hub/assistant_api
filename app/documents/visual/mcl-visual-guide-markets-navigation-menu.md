# Navigation Menu

The navigation menu on the left provides access to various sections like Dashboard, Company, Markets, Departments, and more. This menu facilitates quick navigation across different features and sections of the application, allowing users to manage data and tasks efficiently.

The main navigation menu in MCL Web is your central hub for accessing various features. Each menu item opens a specific configuration area. For example, selecting 'Markets' lets you manage stores and their departments. Other options include 'Departments' for checklist appearances, 'Security' for user roles, 'Questions' for checklist queries, 'Checklists' for managing forms, 'Reports' for generating reports, 'Data Analysis' for result comparisons, 'Tasks' for task management, and 'Photo Management' for handling checklist photos. At the top right, you'll find header icons such as the non-functional Notification Bell, the Help icon for documents, and the Lock Icon for entering lockscreen mode.
