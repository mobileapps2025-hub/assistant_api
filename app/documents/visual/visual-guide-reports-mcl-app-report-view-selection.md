# Report View Selection

On the report view selection screen, users can choose between viewing the report directly within the app in PDF format or emailing it to an address of their choice. This screen provides metadata about the report, including an ID, date, market, checklist, auditor, and type. Two buttons at the bottom allow you to either view the report or send it via email.
