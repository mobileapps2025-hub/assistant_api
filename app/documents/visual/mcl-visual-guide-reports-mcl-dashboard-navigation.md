# Navigating the MCL Dashboard

When you log into the MCL Web UI, you start at the Dashboard. Here, the navigation panel on the left allows you to access various management and analysis modules, such as Reports and Data Analysis. On smaller screens, the navigation menu can be accessed by clicking the hamburger icon in the top-left corner.
