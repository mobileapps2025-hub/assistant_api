# Tag Selection Options

Tags in the system are organized under categories such as Size, Region, Departments, and District Manager. Users can expand each category to select tags that accurately describe the market, such as picking 'Nord' under Region or 'Kitchen Staff' under Ramen Shop. The selected tags will guide how data is filtered and represented on the dashboard.
