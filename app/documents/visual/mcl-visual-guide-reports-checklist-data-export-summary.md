# Checklist Data Export Summary

The 'Datenexport Checklisten-Wertung' offers a summary of checklist executions with a company-wide overview available monthly or weekly. Each row corresponds to an execution event, detailing the store, checklist name, execution date, and responsible user. The 'Wertung' column provides the checklist rating as a percentage, while the 'Expected,' 'Completed,' and 'Missed' columns give insights into the completion status.
