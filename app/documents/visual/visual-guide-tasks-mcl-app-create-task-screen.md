# Create Task Screen

The Create Task screen in the app enables users to manually create new tasks. Users can assign tasks to a specific market and, optionally, to a user. If no user is selected, the task defaults to a pre-assigned user. The screen includes fields to select the market, assign to a user, and set start and due dates. Users can also add comments or photos, and a large 'ADD' button finalizes the task creation.
