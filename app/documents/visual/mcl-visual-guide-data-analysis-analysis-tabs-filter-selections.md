# Understanding Filter Selections in Analysis Tabs

The analysis tabs in the system require specific filter selections based on their purpose. For example, the Chart (Markets) tab necessitates filters for Market and Period to facilitate multi-market comparisons. Different tabs, such as Comparison (Departments), need filters that include multiple markets, a single department, and a period. It's important to select the appropriate filters for each tab to ensure that the charts display correctly. Remember, bars on the charts will represent scores in percentage.
