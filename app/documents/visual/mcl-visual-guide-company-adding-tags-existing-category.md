# Adding Tags to an Existing Category

To add new tags to an existing category, locate the tag category in the list and use the plus icon to include new tags. Remember, each new tag requires the creation of two additional sub-tags. This structure ensures tags are correctly displayed when filtering dashboard charts or during checklist creation.
