# Dashboard KPI Interface Overview

The Dashboard KPI interface provides a quick overview of key performance indicators. It includes four KPI tiles that summarize data for the selected reporting period, such as the current week. These tiles show: **Completed Checklists**, which are informational and not clickable; **Checklists Not Completed**, which offers a clickable tile leading to a detailed list of missed checklists; **Assigned Tasks**, also informational without a drill-down; and **Positive/Negative Answers**, tallying responses in completed checklists without additional navigation.
