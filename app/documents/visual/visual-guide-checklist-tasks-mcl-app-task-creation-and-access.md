# Creating and Accessing Tasks

To create a task, select a responsible user, set the start and due dates, and enter necessary task details. Once completed, press ADD to generate the task and link it to a checklist question. To review existing tasks, use the lens/search icon within the task creation screen to access all related tasks for easy reference.
