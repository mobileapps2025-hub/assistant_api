# Tag Assignment Panel

The Tag Assignment Panel helps organize markets by applying categories such as Region, Store Type, and Franchise Group. Tags are optional but benefit larger retailers by enhancing dashboard filtering and analytics. Within the panel, users can check applicable tags to each market, affecting how KPI summaries are presented based on selected filters.
