# View Roles

The 'View Roles' feature allows you to see all existing role profiles within the application. This screen lists various roles like Auditor, Checklist Administrator, and Company Administrator, along with their associated permissions. You can quickly identify what access and permissions each role has, as indicated by the checkboxes in the role grid.
