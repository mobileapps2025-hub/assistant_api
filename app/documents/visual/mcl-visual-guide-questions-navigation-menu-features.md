# Navigation Menu Features

The MCL Web main navigation menu provides access to various configuration areas such as Markets, Departments, Security, Questions, Checklists, Reports, Data Analysis, Tasks, and Photo Management. Each option allows users to manage different aspects of their environment, like creating checklists or analyzing data. The header icons include a notification bell, which currently has no functionality, a help icon for accessing documentation, and a lock icon to enter lockscreen mode. To proceed with managing questions, select 'Questions' in the menu.
