# Accessing Tasks via Email Notification

When you receive an email notification about a new task, clicking the 'View task' link will redirect you to the task details in the MCL dashboard. This page displays comprehensive task information including the task title, start and due dates, description, and current status such as 'Not started' or 'In progress'. You can also navigate back to the task list or start working on the task immediately.
