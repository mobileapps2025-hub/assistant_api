# Task Completion Email Notification

When a task created by a user is completed, they receive an email notification. This email contains a link allowing the user to view detailed information about the completed task. It helps in keeping track of task statuses and ensuring that users are promptly informed about their tasks' progress.

When a task status changes to 'Completed,' an email is triggered containing task information and a link to view further details. Upon clicking this link, you are taken to the Task Notes view within the Dashboard. Here, you can review the completed task's details and any related comments. You also have the option to reopen the task or return to the task list.
