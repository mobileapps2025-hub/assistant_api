# Report Handling Options After Checklist Completion

After finalizing a checklist, users can manage the resultant report through various options. They select the report type, which includes choices like 'General report', 'Only results', and 'Only Task'. Once chosen, users have the option to send the report via email, open it as a PDF, or share it with selected contacts by using the appropriate icons and the 'Finish' button.
