# Applying Tag Filters in Dashboard

The MCL dashboard provides dropdown menus for applying tag filters, helping users refine data views. Users can select tags from categories such as stores, CL groups, and checklists. Once chosen, these filters influence the visualization of checklist statuses and answers, allowing for a tailored display of data based on the selected criteria.
