# Task Filtering Interface

The task filtering interface allows users to refine their task search by various criteria. You can use the **From Date Selector** and **To Date Selector** to choose a date range for your search. Additionally, the **Due / Create / Start** option helps specify the type of date filter. Use the **Market Field** to focus on tasks relevant to a specific market and the **Assigned To Field** to filter tasks by who is responsible for them. Finally, the **Status Field** lets you filter by task status. Once you've set your parameters, click the **Search Button** to execute the search.
