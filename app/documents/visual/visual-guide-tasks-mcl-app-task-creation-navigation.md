# Task Creation Navigation

To begin creating a new task, navigate to the **Task creation** screen from the **Tasks Overview**. This screen provides options for detailing a new task. Located on the **Task Overview** child screen, it guides users through the creation process. Ensure you have completed the necessary fields before saving the new task.
