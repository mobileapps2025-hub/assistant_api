# Task Creation Form

The Task Creation Form in the MCL app lets you customize your task by assigning it to a user and setting optional start and due dates. You must include a task description if no photo is uploaded. Additionally, you can toggle between adding a photo or a comment to provide more context. Clicking the Add Button will save and create your task.
