# Task Configuration Form

The task configuration screen allows users to document the necessary actions by specifying details such as the responsible user, active timeframe, and adding relevant notes or photos. Key fields include 'Who Field' to assign the task, 'Start Date' and 'Due Date' for timing, and 'Task Description Field' for notes. Users can use the 'Voice Input Icon' for text entry and 'Photo Upload Area' for evidence. The 'Add Button' finalizes the task creation. Note that either a photo or comment is mandatory, and adding a start date requires a due date as well.
