# Overview of Notification Types

This section provides an overview of notification types within the MCL system. Notifications in MCL are categorized by type and follow a structured flow from notification to potential interaction, and result. Push and task emails require user actions, while checklist reports offer detailed results. Analytics emails focus on delivering summaries.
