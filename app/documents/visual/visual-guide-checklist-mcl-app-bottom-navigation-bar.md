# Bottom Navigation Bar

The bottom navigation bar provides quick access to the app’s main functions through a series of icons. The Checklist icon opens the store overview, and selecting a store will further open its checklist overview. The Book icon lets you access saved checklist reports, while the Clock icon opens a tasks overview. Finally, the User icon is used for app configuration settings.
