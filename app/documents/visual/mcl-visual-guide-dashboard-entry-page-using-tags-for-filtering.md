# Using Tags for Filtering

Tags in the MCL app allow users to categorize and filter markets using custom criteria. These tags are organized under distinct categories and can be applied to markets and checklists. There are three types of tags available: Store/Store Group Tags, which categorize individual stores or store groups; Checklist Group Tags, which organize multiple checklists under a common classification; and Individual Checklist Tags, tailored to specific checklists. When tags are selected, they dynamically adjust the dashboard charts according to the chosen filters.
