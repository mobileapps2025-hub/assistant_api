# Departments Overview

The Departments Overview screen displays all departments associated with the selected company. You can use the Search Bar to filter through departments by entering text. The Departments Table lists each department, showing their name and an optional description. To create a new department, click the “+” icon. To modify an existing department, select the Pencil icon. You can delete a department using the Trash icon.
