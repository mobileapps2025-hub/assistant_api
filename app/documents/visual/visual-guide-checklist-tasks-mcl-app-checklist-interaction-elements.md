# Checklist Interaction Elements

The checklist interface provides various elements to facilitate user interaction. You can monitor your progress using the 'Question Progress' which shows your current position and the total number of questions. Answering questions is straightforward with 'YES' or 'NO' buttons. For additional context, the 'Notes Field' allows writing or recording comments, while 'Photo Icons' let you attach images. To communicate verbally, the 'Voice Recording Icon' records your voice and translates it into text. If a question is irrelevant, mark it with the 'N/A Toggle'.
