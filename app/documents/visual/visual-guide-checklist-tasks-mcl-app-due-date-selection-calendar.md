# Due Date Selection in Calendar

In the 'Task configuration - Due Date Selection' screen, you can choose a start date for your task using the calendar pop-up. This interface allows you to navigate between months using the month navigation arrows. Once a date is selected, it will be highlighted in blue, indicating your chosen start date. Remember, after selecting a start date, setting a due date becomes mandatory.
