# Role Configuration

The Roles section in the MCL interface is crucial for defining user permission profiles across the MCL Web and App. By configuring roles, you determine which features users can access and which menu sections are visible to them, under the Security Sub-Menu. This configuration ensures that users have access appropriate to their responsibilities.
