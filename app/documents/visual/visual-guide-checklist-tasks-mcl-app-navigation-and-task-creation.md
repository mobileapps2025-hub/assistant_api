# Navigation and Task Creation

To navigate through the checklist, use the 'Navigation Arrows' to move between questions. If you encounter an issue that needs action, you can create a task without leaving the checklist by selecting the 'Calendar Icon' in the center, which opens the task creation screen. This seamless transition ensures you can assign tasks immediately as you encounter issues during an audit.
