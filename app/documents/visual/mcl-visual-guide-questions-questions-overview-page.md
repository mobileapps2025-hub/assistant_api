# Questions Overview Page

The Questions Overview Page, accessed from the main navigation menu, is where you can view, create, edit, or delete questions used in checklists. This section includes options for managing all aspects related to questions, ensuring your checklists are updated with relevant inquiries. From here, you can proceed to screens for creating, editing, or deleting questions.

The Questions Overview page allows users to manage checklist questions effectively. You can search for questions using the search bar or filter them by department using the Department Filter Dropdown. To create, edit, or delete questions, use the corresponding action icons: '+' for creating, the pencil icon for editing, and the trash icon for deleting. The page displays a table with columns for question text, additional information, and question type.
