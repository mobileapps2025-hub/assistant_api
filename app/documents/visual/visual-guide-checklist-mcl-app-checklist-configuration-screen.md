# Checklist Configuration Screen

The Checklist Configuration screen allows users to prepare a checklist before execution. This step is crucial to ensure the checklist aligns with the specific conditions of the store. Users need to specify the Market, Checklist Name, and Auditor details. Additionally, the Reason for the checklist can be selected as either 'Routine' or 'Special'. Departments involved in the checklist execution are also selected on this screen. Once all fields are completed, the user can tap 'Start Checklist' to proceed.
