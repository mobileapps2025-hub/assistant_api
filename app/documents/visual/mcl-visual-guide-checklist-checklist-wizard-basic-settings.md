# Configuring the Checklist Wizard's Basic Settings

In the Checklist Wizard, the Basic Settings panel is crucial. Make sure to name your checklist and define its periodicity, choosing from options like daily, weekly, or yearly. Completing these fields is necessary before proceeding.
