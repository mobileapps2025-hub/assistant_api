# Filtered Task List

The Filtered Task List screen displays tasks that are new or due. Each task entry shows the task title, due date, and current status, such as 'Not started.' You can use the filter option to adjust which tasks are displayed. This interface allows users to easily access the most pertinent tasks at a glance.
