# Tasks Sub-Menu

Within the main navigation of MCL Web, selecting 'Tasks' allows you to access the Tasks Overview. This section provides further options like viewing, editing, and assigning tasks. The screen transitions from the main menu to provide dedicated task management, enabling users to maintain their task lists efficiently.
