# Bottom Navigation Bar

The bottom navigation bar offers quick access to essential features of the app. The checklist icon opens the store overview, and selecting a store will navigate to a checklist overview. Tapping the book icon allows access to checklist reports on the device, while the clock icon opens the tasks overview. The user icon is used to access the app configuration settings.
