# Navigating the Task List

The task list screen allows users to view and manage existing tasks. A back button is provided to return to task creation. The list is scrollable and displays all tasks previously created, helping prevent duplicates. Users can swipe to either delete or edit tasks, ensuring efficient task management. This feature operates independently, so no changes are made unless a task is actively selected.
