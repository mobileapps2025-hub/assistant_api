# Main Navigation Menu

Upon logging in, users are directed to the MCL Dashboard interface. The main menu serves as the root navigation point, providing access to various configuration areas such as Security and Checklists. Key icons in the header include a notification bell, a help icon, and a lock icon for entering lockscreen mode. From the dashboard, you can view metrics like completed checklists and access other functions like data analysis and task management.
