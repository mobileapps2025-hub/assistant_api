# New Task Assignment Email Notification

When a task is assigned, users receive an email notification that includes essential information about the task. The email displays the task title, due date, the user it's assigned to, and a direct link to the task. This email is automatically triggered whenever a new task is assigned, facilitating immediate access and review.
