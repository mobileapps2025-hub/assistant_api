# User Management Interface

The User Management interface allows administrators to create and maintain user accounts within the system. You can add a new user by clicking the plus icon visible on the screen, which opens fields to enter user details such as Name, Email, and Role. You can also edit existing user details or delete a user if they are no longer needed. The interface also supports actions like assigning roles to users, which helps manage their permissions within the system.
