# Checklist Overview and Creation

The main interface displays your existing checklists, providing a summary of each list’s configuration, including frequency and departments involved. To start creating a new checklist, click the '+' icon which opens the Checklist Wizard. This intuitive tool guides you through the creation process step-by-step.
