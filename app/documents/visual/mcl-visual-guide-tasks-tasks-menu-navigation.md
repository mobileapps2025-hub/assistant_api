# Navigating the Tasks Menu

The Tasks menu provides two options: To Do List and New Task. Selecting 'To Do List' allows users to view, filter, edit, complete, or delete existing tasks. Alternatively, choosing 'New Task' lets users create a task from scratch. Use the interface to navigate to the desired function.
