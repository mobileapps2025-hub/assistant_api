# Completed Checklists Overview

The 'Completed Checklists' bar graph represents each market with a distinct bar. Each color within a bar signifies a different completed checklist, with the possibility of multiple colors indicating multiple checklists. By hovering over these bars, a pop-up detail will show the exact number of each type of completed checklist within the selected timeframe.
