# PDF Report View

The PDF Report View offers a comprehensive display of the checklist results in a structured document, mirroring the official report format used for exporting on the web. It includes company and market details, checklist name and report ID, start and completion times, auditor's name, as well as an overall rating. The checklist questions and corresponding answers are presented clearly with color indicators for different answer statuses. This read-only report ensures standardized documentation, and users can opt to email the report rather than viewing it within the app.
