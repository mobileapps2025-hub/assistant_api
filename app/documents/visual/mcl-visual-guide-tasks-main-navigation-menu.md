# Main Navigation Menu

Upon logging into the MCL system, users are directed to the Dashboard, where they can efficiently navigate through various modules using the navigation panel on the left. This panel, outlined in red in the screenshot, provides access to essential features like Checklists, Tasks, and Data Analysis. Users on smaller screens will need to click the hamburger icon at the top left to reveal the menu.
