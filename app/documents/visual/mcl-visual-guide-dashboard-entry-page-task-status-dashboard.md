# Understanding Task Status on the Dashboard

The 'Status of Tasks with Due Date' section on the dashboard provides a visual representation of tasks due within a selected period. Each bar represents different task statuses using color codes: green for completed tasks, red for missed tasks, and yellow for open tasks that are due but not finished. Clicking on any segment of a bar gives you a detailed list of the tasks. If there’s no bar visible for a market, it means no tasks are due this week.
