# Creating a New Checklist Question

To create a new checklist question, click the "+" icon to begin. Enter the required question text and optional additional information. If needed, upload a PDF attachment with a maximum size of 3MB. Choose the question type, such as Yes/No, grading, or free text. Define when comments or photos are required. Set any additional behaviors, like triggering tasks, and establish positive answers. Select one or more departments to assign the question to before saving your setup.
