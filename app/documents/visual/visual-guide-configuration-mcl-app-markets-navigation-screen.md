# Markets Navigation Screen

Upon logging into the MCL app, users are directed to the 'Markets' screen. This screen serves as a hub where users can access various markets, identified by names such as 'Markt 1' and 'Van’s Ramen.' From here, users can complete checklists or tasks. The icons encased in the red box at the bottom of the screen provide navigation to other essential functions of the app.
