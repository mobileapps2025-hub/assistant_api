# Interactive Data Charts

The platform features interactive charts that allow users to click and view underlying data, offering a detailed analysis experience. Exact score values are displayed on labels directly, such as 61.4% and 55.0%. This functionality is used for comparing market performances, analyzing trends, and visual communication of performance results.
