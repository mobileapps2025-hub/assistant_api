# Main Navigation Menu

When users log in to the MCL system, they arrive at the MCL Dashboard. On the right side, there's a navigation panel (highlighted in red), granting access to all system management and analysis modules. If you're on a smaller screen, access the full menu by clicking the hamburger icon at the top.
