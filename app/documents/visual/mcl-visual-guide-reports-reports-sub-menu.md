# Reports Sub-Menu

Within the MCL navigation menu, selecting 'Reports' opens the Reports overview. This section is divided into 'Ad Hoc Reports' and 'Canned Reports,' enabling users to generate and access various report types. This functionality is key for analyzing checklist data and exporting it in formats like PDF and Excel.
