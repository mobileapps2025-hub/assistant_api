# Task Execution: Adding a Note

To add a note to a task, click the "+" icon in the task row. This will bring you to a new screen where you can click "Create a note". Enter your note details in the 'Task description' field, optionally upload an image, and decide if you want to mark the task as closed by checking the box. After you're done, click the save icon to store your changes.
