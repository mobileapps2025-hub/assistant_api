# Navigating to Photo Management

Upon logging into the MCL system, users are directed to the Dashboard. To access the Photo Management module, use the navigation panel located on the left side of the interface, highlighted in a red box. If the panel is not visible due to a smaller screen size, click the hamburger icon in the top-left corner to reveal the menu.

The MCL Web application provides a centralized navigation menu where you can access various configurations. To browse and manage the uploaded checklist photos, select 'Photo Management' from the main menu. This section offers tools to efficiently handle checklist-related photos.
