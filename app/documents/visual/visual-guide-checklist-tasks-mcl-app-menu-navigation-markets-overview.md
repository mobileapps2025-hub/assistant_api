# Menu Navigation: Markets Overview

Upon logging into the MCL App, the first screen visible is the 'Markets' overview. Here, users can select various market options listed on the screen to proceed with tasks or explore different app functions. The bottom navigation bar, highlighted, provides quick access to other key areas of the app for smooth navigation.
