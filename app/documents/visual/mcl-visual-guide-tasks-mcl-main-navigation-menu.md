# MCL Main Navigation Menu

The MCL Web platform's main navigation menu allows you to access different configuration areas. Key features include managing markets, departments, security settings, questions, and checklists. You can also generate reports, analyze data, and manage tasks and photos. The header icons provide quick access to notifications, help, and lockscreen mode. To proceed to specific tasks such as checklist management, select the corresponding section from the menu.
