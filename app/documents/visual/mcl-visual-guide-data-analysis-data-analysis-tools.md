# Data Analysis Tools

The **Data Analysis** section in the MCL app provides two key tools for analyzing checklist data: **Result Analysis** and **Comparative Analysis**. Result Analysis allows you to perform table-based evaluations, focusing on checklist answer ratios. Comparative Analysis offers visual, interactive charts for comparing checklist percentages according to various criteria. These tools help assess performance across different markets and departments.
