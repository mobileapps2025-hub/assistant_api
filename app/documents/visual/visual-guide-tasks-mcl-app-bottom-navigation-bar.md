# Bottom Navigation Bar

The bottom navigation bar in the app provides quick access to key functions. It includes icons for various features: the Checklist Icon opens the store overview, the Book Icon gives access to checklist reports, the Clock Icon opens the tasks overview, and the User Icon allows you to configure the app.
