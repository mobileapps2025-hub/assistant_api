# Dashboard Answer Distribution

The dashboard provides a visual representation of the distribution of answer types across completed checklists for various markets. It uses color-coded bars to show the percentage and absolute number of each answer type: green for positive, red for negative, yellow for not applicable (N/A), and blue for not counted (free text answers). If a bar is not displayed, it indicates that there are no completed checklists for that market.
