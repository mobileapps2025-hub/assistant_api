# Configuring Report Checklist Options

In the Report Checklist view, you have three toggle options to customize the report details you wish to include. Selecting **General report** will send a complete report with all answers, notes, images, and tasks. Choosing **Only results** includes answers, notes, and images, but omits tasks. If you select **Only task list**, your report will solely consist of tasks created in the checklist. After selecting the desired option, use the icons below to share or download the report.
