# Synchronization Interface

The synchronization interface displays the last date and time when synchronization occurred via the 'Synchronize Timestamp'. It also shows the number of reports pending upload under 'Reports Not Yet Synchronized'. The 'Synchronize Now Button' currently has no functionality.
