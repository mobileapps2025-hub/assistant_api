# Permissions Column Overview

The permissions column in the MCL app allows you to manage what users in a specific role can do. For example, checking 'Dashboard' lets users access and view key performance indicators, while 'Tasks assigned to Market' allows task assignments at the market level. If unchecked, users' access to these features will be restricted.
